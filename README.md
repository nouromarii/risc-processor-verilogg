## RISC Processor Design (Verilog)

A computer architecture project implementing a simplified pipelined RISC processor.

##  Overview
This project focuses on designing and simulating a multi-stage pipelined processor using Verilog.

##  Components
- Fetch Stage
- Decode Stage
- Execute Stage
- Memory Stage
- Write Back Stage

##  Modules Implemented
- Instruction Memory
- Register File
- Main Control Unit
- ALU
- Pipeline Registers

##  Features
- Instruction execution through pipeline stages
- Control signal generation
- Basic hazard handling

## Technologies
- Verilog
- Digital Design
- Computer Architecture concepts

## Learning Outcome
Gained hands-on experience in processor design, pipelining, and hardware-level implementation.
